# PROJECT: INTERACTIVE TODO LIST

Goals:
- Let users add tasks dynamically
- Let users remove tasks by clicking on them
- Use DOM manipulation to update the UI (User Interface)

What we did:
- Used createElement to create the list item.
- Used aapendChild to append the item to the list.
- Used addEventListener to add the removeChild.

Final thoughts:
- DOM allows JS to interact dynamically with HTML.
- selecting elements is done by using .getElementById and querySelector.
- Modifying elements is possible using text content, innerHTML and style.
- Creating and removing elements is done by using createElement, appendChild and removeChild.
