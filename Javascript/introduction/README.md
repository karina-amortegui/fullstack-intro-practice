# OVERVIEW OF JAVASCRIPT

## WHAT IS JAVASCRIPT?

- a programming language used to make websites interactive.
- it add dynamism to web pages and enables:
- form validation
- pop-ups and alerts
- image sliders
- interactive buttons
- dynamic content updates

## WHY IS JAVASCRIPT IMPORTANT?

- it works in web browsers/runs inside web browsers (chrome, firefox, and edge) without needing additional installations.
- it is used everywhere: modern web appls, mobile apps, and even backend systems (Node.js) rely on JavaScript.
- Beginner-friendly: easy to start learning with just a browser and a text editor.

## JavaScript in Webdevelopment

A web page consists of three main technologies:
    1. HTML (hypertext markup language) - creates the structure of the page.
    2. CSS (cascading style sheets) - adds styles and layouts.
    3. JavaScript - adds interactivity and functionality.

## HOW TO ADD JAVASCRIPT TO A WEBPAGE

Two main ways to use JavaScript:
    1. in-line (inside HTML)
    2. external JavaScript files with a .js.

## KEY TAKEAWAYS

- Javascript makes web pages interactive.
- It works with HTML and CSS to create dynamic content.
- Javascript can be written inline or externally.
- Use console.log to debug and test code in the browser's console.
