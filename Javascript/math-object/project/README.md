# PROJECT: RANDOM DICE ROLLER & GAME SCORE TRACKER
Goals:
- Simulate rolling a dice 1-6 using Math.random
- Keep track of how many times each number appears
- Let the user roll the dice multiple times and display results

## What is included in this project:
- Math.random to generate random dice rolls.
- Uses an object, the score tracker, to store the count of each number.
- Loops: for loop to roll the dice multiple times and uses for...in to display score results.
