## SETTING TEXT COLOR

### use the color property to change text color.
### hex colors (rgb): ff0000 (red) 00ff00 (green) 0000ff (blue)
### hex colors: values 0-9, letters a,b,c,d,e,f,g or A,B,C,D,E,F,G

### differnt ways to define colors:

#### name: red, green, blue
#### hex codes: #301583, #e42d0c, #117778
#### rgb (red,green,blue): rgb(0,0,255), rgb(19,45,0) 
#### rgba (controls opacity/transparency): 4th value  (0-1) added at the end of rgb sequence -- rgba(17, 17, 201, 0.8)

## IMPLEMENTING BACKGROUND IMAGES

## BACKGROUND POSITIONING

### use background position to align images.
### values: 
#### left top (aligns top left)
#### center center (centers the image)
#### right bottom (aligns bottom right)

## BACKGROUND SIZING

### defines how the image fits withihn an element.
### values: auto, cover, and maintain.
### cover fills the space according to whatever the space size is.
### contain keeps the image fully visible and may repeat when space is expnded.  

## BACKGROUND REPEATING

### controls image tiling.
### repeating values:
#### repeat: repeats in both directions
#### repeat X: repeats horizontally
#### repeat Y: repeats vertically
#### no repeat: no repition

## BACKGROUND ATTACHEMENT

### controls whether the background images scroll with the page.
### in CSS, background attachment has values:
#### scroll (moves with page scroll)
#### fixed (stays in one place)
#### local (moves within the elements' content)

## BACKGROUND LINEAR GRADIENTS

### types:
#### linear gradient (from left to right)
#### radio gradient (like a circle)
#### gradients improve modern design aesthetics
