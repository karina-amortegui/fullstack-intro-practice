# CSS FLEXBOX FOR RESPONSIVE LAYOUTS

## flexbox is a css layout model that allows elements to be easily aligned, spaced, and resized within a container.
## makes designing responsive layouts much easier.

## WHAT IS FLEXBOX?

### a one-dimensional layout system that arranges elements horizontally (in rows), or vertically (in columns).

## WHY USE FLEXBOX?

### it is easier to: align elements, centering, spacing, distributing
### adapts to different screen sizes
### great for responsive design
### reduces the need for floats and positioning hacks

## FLEXBOX AXES

### row (default): left to right on the main axis
### column: top to bottom

## ENABLING FLEXBOX
### to activate flexbox, set display Flex on the parent container.

## FLEXBOX PROPERTIES

### direction
#### row, row-reverse, column, column-reverse

### justify 
#### space-between, space-around, space-evenly, flex-start (default), flex-end, center

### align 
#### controls vertical alignment on default
#### flexstart (default) - puts boxes to the top
#### flexend - puts boxes to the bottom
#### center - brings them to the middle
#### stretch (default) - extends vertical length
### baseline

### align-self
#### overrides the alignment for a single item
#### aligns an individual item differently

## WRAPING ITEMS (flex-wrap)

### by default items stay in one row
### use flexwrap to allow items to wrap when needed 

### flexwrap has 3 values:
#### no-wrap (default)
#### wrap
#### wrap reverse
