# CSS GRID TO DESIGN (2-D Layouts)

## a powerful layout system that allows you to design two-dimensional layouts with items positioned in rows and columns.
## more flexible than flexbox when working with complex page layouts.

## WHY USE CSS GRID?

### it creates complex layouts easily
### better for structuring for full webpages
### precise control over placement

## ENABLING CSS GRID

### activate css grid
### set display grid on a parent container

## DEFINING COLUMNS & ROWS

### use grid template columns and rows to creat a structured grid.
### using FR or flexible fraction, instead of pixels. use FR for dynamic resizing.

## PLACING ITEMS IN THE GRID

### use grid column and grid row to position elements.

## GRID GAPS-SPACING

### use gaps to add space between grid items.

## GRID ITEM ALIGNMENT

### horizontal alignment (justify items):
#### start - aligns to the left
#### center - aligns in the center
#### right - aligns to the right


### vertical alignment 
#### start - top
#### center - center
#### end - bottom
