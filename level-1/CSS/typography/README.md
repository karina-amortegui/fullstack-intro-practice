# TYPOGRAPHY

## one of the most important aspects of web design.
## determines how text looks, feels, and behaves on a webpage.

## STYLING FONTS

### font family (choosing the right fonts)
### font size (scaling text appropriately)
### font weight & style (bold, italic, etc.)
### text alignment & spacing (line height, letter spacing)
### text transformation & decoration (uppercase, underline, etc.)
### google fonts (importing custom fonts)

### serif fonts have small strokes at the end of the letters.
### sans serif fonts are clean and modern.
### monospace fonts have envenly spaced characters --- great for coding.

## CHANGING FONT SIZE

### the font size property defines text size in pixels (px, m, rem, %, and v port units: view width (vw) and view height (vh)).

## FONT WEIGHT & FONT STYLE

### font weight refers to the boldness of the text.
#### values range from 100 (thin) to 900 (extra bold).
### font style refers to whether it is italicized.

## TEXT ALIGNMENT 

### left, center, right.

## LINE HEIGHT & SPACING

### controls the space between lines. size 1 is normal.
### letter spacing: the space between letters.

## TEXT TRANSFORMATION 

### uppercase, lowercase, or capitalize.
### can do with CSS, just add style tag in .html

## TEXT DECORATION

### overline, underline, and strikethrough.

## IMPORTING GOOGLE FONTS

### can load custom fonts from Google Fonts for options beyond build-in HTML fonts.
